# Project Overview

A modern, responsive, and beautifully animated personal portfolio website built from scratch. The site uses a premium glassmorphism aesthetic with a dark-mode-first design, fluid Framer Motion animations, and a component-first architecture.

## Tech Stack

| Category       | Technology                    |
|----------------|-------------------------------|
| Framework      | React 19 + Vite               |
| Styling        | Tailwind CSS v3               |
| Animations     | Framer Motion                 |
| Icons          | React Icons (Feather + SI)    |
| Typography     | Google Fonts (Inter, Poppins) |
| Deployment     | Vercel                        |

## Key Features

- Glassmorphism UI with frosted-glass cards and translucent surfaces
- Parallax scrolling in the Hero section with multi-layered depth
- Starfield particle animation in the Hero background
- Staggered on-scroll reveals across all sections
- Spring-based skill card animations
- Gradient text, glow borders, and ambient background orbs
- Fully responsive across mobile, tablet, and desktop viewports
- Accessible via prefers-reduced-motion support
- Custom gradient scrollbar
- Animated hamburger-to-X mobile navigation
